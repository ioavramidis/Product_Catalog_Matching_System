This is an automated matching system that maps unstructured product descriptions to the correct catalogue SKUs, demonstrating both technical proficiency and business acumen.
Dataset: Unstructured Descriptions (~250 entries)
Natural language sales enquiries, marketplace listings, customer requests from various channels.
Dataset: Product Catalogue (~800 entries)
Official SKU database with fields: SKU, Product_Name, Category, Colour, Size, Material, Features, Season, Brand, Price.
