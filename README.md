This is an automated matching system that maps unstructured product descriptions to the correct catalog SKUs, demonstrating both technical proficiency and business acumen.

**Dataset Unstructured Descriptions (~250 entries)**
Natural language sales inquiries, marketplace listings, customer requests from various channels.

**Dataset: Product Catalog (~800 entries)**
Official SKU database with fields: SKU, Product_Name, Category, Color, Size, Material, Features, Season, Brand, Price.
